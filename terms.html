Termistö

Termi Selite
(Action) Rule Yksittäinen liiketoimintasääntö, mikä sisältää paikallisia muuttujamäärittelyjä (valinnainen osa), ehtolauseita (valinnainen osa) ja toimintoja (pakollinen osa).
Business Object Model (BOM) Sääntökoneohjelman tarvitsemia luokkamäärityksiä ja dynaamisia arvojoukkoja. Luodaan Rule Designerillä, ei mahdollisuutta muokata Decision Centerissä.
Branch Sääntökoneohjelman kehityshaara. Pääkehityshaara on nimeltään 'main'. Muutokset muista kehityshaarasta toiseen voidaan yhdistää 'merge' toiminnolla.
Decision Operation Päätösoperaatioon määritellään syöte- ja tulosmuuttujat ja suoritettavat säännöt. Yhdessä sääntökonepalvelussa voi olla monta päätösoperaatiota.
Decision Service Sääntökonepalvelu, mikä koostuu yhdestä päätason sääntöprojektista ja mahdollisista aliprojekteista.
Decision Table Päätöstaulu, muistuttaa Excel-taulukkoa, jokainen rivi on oma liiketoimintasääntö.
Deployment Configuration Konfiguraatio, minkä avulla sääntökonepalvelu viedään suoritettavaksi ohjelmaksi palvelimelle.
Dynamic Domain Dynaaminen arvojoukko, mikä määritellään esim. Excel-tiedoston perusteella.
Execution Object Model (XOM) Räätälöityä ohjelmakoodia (yleensä Java), mihin voidaan viitata BOMin kautta. Luodaan Rule Designerillä, ei mahdollisuutta muokata Decision Centerissä.
Priority Numeroarvo, mikä määrittää yksittäisen säännön tai sääntötaulun suoritusjärjestyksen suhteessa muihin valittuihin sääntöihin. Mitä pienempi arvo on, sitä myöhemmin sääntö suoritetaan, ja sitä "vahvempi" kyseinen sääntö on.
Rule Designer Kehittäjän työkalu, käytännössä Eclipse ohjelmointiympäristöön asennettava laajennos.
Ruleflow Vuokaavio, mikä ohjaa sääntöjen suorittamista.
Snapshot Sääntökonepalvelun tallennettu tila, voidaan luoda manuaalisesti tai esimerkiksi deploymentin yhteydessä.
Test Suite Testipaketti yhden päätösoperaation testaamiseksi. Testipaketeissa käytetään Excel-skenaariotiedostoja.
Variable Set Joukko muuttujia, joita käytetään sääntöohjelmassa. Jokaisella muuttujalla on tekninen nimi, tietotyyppi, verbalisointi ja mahdollinen alkuarvo.
Verbalization Nimi millä muuttujaan, luokan jäseneen tai funktioon viitataan säännöissä.
