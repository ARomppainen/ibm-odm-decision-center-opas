Toimintasäännöt (Action Rule)

Säännön rakenne definitions (voidaan jättää pois) sääntökohtaisten muuttujien määrittely if (voidaan jättää pois, tällöin sääntö suoritetaan aina) ehtolause then (ainoa pakollinen osa) toiminnot else vaihtoehtoiset toiminnot Täydennysmenu avautuu painamalla Ctrl + Space. Yleisimmät ehtolauseen rakenteet: muuttuja is arvo muuttuja is one of { arvo1, arvo2, ... }
